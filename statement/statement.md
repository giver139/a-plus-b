# A + B

## Problem Description

Given two integer $A,B$ , please calculate the number $A+B$ .

## Input Format

Input one line containing two integer $A$ and $B$ separated by space.
$-2147483648\le A,B\le 2147483647$

## Output Format

Output one line containing an integer $A+B$.

## Sample Input 1

```
1 2
```

## Sample Output 1

```
3
```

## Sample Input 2

```
-5 0
```

## Sample Output 2

```
-5
```
